О проекте

Weather Dashboard

Проект разработан для отображения актуальной информации о погоде в любом городе мира. Используются данные из надежных источников прогноза погоды и геолокации.

Возможности

  • Текущая погода
  • Прогноз на 10(3) дней
  • Интерактивная карта
  • Поиск по городам

Технологии

  • Spring Boot
  • Яндекс.Карты API
  • WeatherAPI
  • JavaScript
